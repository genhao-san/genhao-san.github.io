Linux系统命令

发表于2025-11-01|更新于2025-11-09|入门
|总字数:831|阅读时长:3分钟|浏览量:

一、文件/目录操作

  • ls ：列出目录内容（ ls -l 查看详细信息， ls -a 显示隐藏文件）。
  • cd ：切换目录（ cd / 到根目录， cd .. 返回上一级， cd ~ 到当前用户家目录）。
  • pwd ：显示当前所在目录的绝对路径。
  • cat ：读取并打印文件内容（ cat /etc/passwd 读取系统用户文件）。
  • tac ：反向打印文件内容（绕过 cat 过滤时用）。
  • cp ：复制文件/目录（ cp test.txt /tmp/ 复制文件到tmp目录）。
  • mv ：移动/重命名文件（ mv old.txt new.txt 重命名， mv file /dir/ 移动文件）。
  • rm ：删除文件/目录（ rm test.txt 删文件， rm -rf /dir/ 强制删除目录及内容，慎用）。
  • mkdir ：创建目录（ mkdir test 创建单目录， mkdir -p a/b/c 创建多级目录）。
  • rmdir ：删除空目录（仅能删空目录，非空需用 rm -rf ）。

二、文件内容查看（CTF高频）

  • more / less ：分页查看大文件（ more /var/log/nginx/access.log ， less 支持上下翻页）。
  • head / tail ：查看文件开头/结尾（ head -5 test.txt 看前5行， tail -f log.txt 实时监控日志）。
  • grep ：搜索文件内容（ grep “flag” test.txt 搜索含“flag”的行， grep -r “key” /dir/ 递归搜索目录）。
  • nl ：显示文件内容并标注行号（ nl flag.php 查看PHP源码行号）。
  • rev ：反转每行字符顺序（ rev test.txt 将“abc”变成“cba”）。

三、系统信息查看（信息收集）

  • uname ：查看系统内核信息（ uname -a 显示完整内核版本、主机名等）。
  • whoami ：显示当前登录的用户名（CTF中确认权限常用）。
  • id ：查看当前用户的UID、GID及所属组（ id 显示详细信息）。
  • ps ：查看进程信息（ ps -aux 显示所有进程， ps -ef | grep php 查找PHP相关进程）。
  • netstat ：查看网络连接（ netstat -tuln 显示监听的端口， netstat -an 显示所有连接）。
  • ifconfig / ip addr ：查看网卡IP地址（ ifconfig 旧版， ip addr 新版系统常用）。
  • df ：查看磁盘空间使用情况（ df -h 以人类可读格式显示）。

四、权限管理（CTF提权相关）

  • chmod ：修改文件/目录权限（ chmod 777 test.txt 给所有用户读、写、执行权限， chmod +x shell.sh 给脚本执行权限）。
  • chown ：修改文件/目录所有者（ chown root:root test.txt 将所有者改为root用户和root组）。
  • sudo ：以其他用户（默认root）身份执行命令（ sudo cat /root/flag.txt 用root权限读flag）。

五、压缩/解压（处理备份文件）

  • zip / unzip ：压缩/解压zip文件（ zip -r www.zip /var/www/ 压缩目录， unzip www.zip 解压文件）。
  • tar ：压缩/解压tar/tar.gz文件（ tar -zcvf www.tar.gz /var/www/ 压缩， tar -zxvf www.tar.gz 解压）。

六、其他实用命令（CTF常用）

  • find ：查找文件/目录（ find / -name “flag*” 从根目录搜索以“flag”开头的文件）。
  • echo ：输出字符串或变量（ echo “hello” 输出内容， echo $PATH 显示环境变量）。
  • base64 ：Base64编码/解码（ echo “flag” | base64 编码， echo “ZmxhZw==” | base64 -d 解码）。
  • ping ：测试网络连通性（ ping 192.168.1.1 测试与目标IP的连接）。
  • wget / curl ：下载文件（ wget http://xxx.com/shell.sh 下载脚本， curl http://xxx.com/flag 访问网页内容）。
